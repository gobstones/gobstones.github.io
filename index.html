Gobstones es un lenguaje pensado y diseñado para la enseñanza introductoria a la programación.

¡Probá Gobstones Web ahora!

Programá con bloques
Gobstones JR permite que la tarea de escribir código sea tan simple como crear y mover bloques.

Programá con texto
Gobstones SR permite a los estudiantes aprender conceptos más avanzados utilizando un lenguaje formal.

Herramientas para docentes
Gobstones Teacher permite a los docentes crear guías de ejercicios y distribuirlos fácilmente.

Versiones Desktop
Gobstones es software libre y tiene versiones de escritorio que pueden descargarse desde esta misma página.
